# 📌 VERY IMP NOTES

### 1. `candidates.jsonl` is intentionally **not included** in this repository.

The original candidate dataset was very large and was provided as part of the hackathon challenge. It has been removed from `Project/data/` to keep the repository lightweight. Place the dataset back into the `data/` folder before running the project.

### 2. `ranked_candidates.json` has been cleaned.

The original file contained all ranked candidates and was several hundred megabytes in size due to embeddings and ranking information. It has been removed/cleaned from the repository to reduce GitHub repository size. The file can be regenerated by executing `main.py`.

# 🚀 AI Resume Parser & Candidate Ranking System

An AI-powered Resume Parser and Candidate Ranking System developed for the Resume Parser and Ranker Hackathon.

The system automatically parses a Job Description (JD), extracts hiring requirements, processes candidate profiles, performs semantic matching using transformer embeddings, computes weighted relevance scores, and generates a ranked list of the best candidates with explainable reasoning.

---

# ✨ Features

- 📄 Automatic Job Description Parsing
- 👤 Candidate Resume Parsing
- 🧹 Candidate Data Normalization
- 🧠 Semantic Matching using Sentence Transformers
- 🎯 Skill, Experience & Embedding-based Matching
- 📊 Weighted Candidate Scoring
- 🏆 Candidate Ranking
- 💬 Explainable Recommendation Generation
- 📑 CSV Submission Generation

# 🧠 Ranking Signals

The ranking engine combines multiple candidate signals:

- Technical Skills
- Years of Experience
- Semantic Similarity
- Behaviour Signals
- Company Preference
- Preferred Location

Each signal contributes to the final weighted ranking score.
